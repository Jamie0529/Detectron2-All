# Detectron2-All是什么?
  Detectron2-All是一个基于Detectron2的神经网络快速搭建，其与Detectron2的区别在于：Detectron2只内置了目标检测，图像分割等领域的算法和数据集，面向的领域比较窄，而Detectron2-All致力于内置所有常用机器学习算法（以深度学习为主）和所有常用数据集，包括但不限于分类、回归、小样本、元学习等领域。
